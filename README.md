# 🇻🇳 DiscoverVN — Vietnam Travel Guide

> Final Project for IBM Full Stack JavaScript Developer Course on Coursera

A responsive, multi-page travel guide website built with HTML, CSS, and vanilla JavaScript — showcasing Vietnam's best beaches, temples, and travel routes.

**🌐 Live Site:** `https://TungTavv.github.io/vietnam-travel`

---

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — hero section, introduction, and three destination recommendations |
| `about.html` | About Us — mission, story, and team |
| `contact.html` | Contact Us — email contact form |

vietnam-travel/

## Features

- Fixed navigation bar with active link highlighting and mobile hamburger menu
- Beach recommendation — Phú Quốc Island (2 images)
- Temple recommendation — Mỹ Sơn Sanctuary (2 images)
- Country route recommendation — North-to-South Vietnam (2 images)
- Contact form with client-side submission handling
- Scroll-reveal animations via IntersectionObserver
- Fully responsive down to mobile (≥ 320px)
- `prefers-reduced-motion` support

## Built With

- HTML5 / CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts — Playfair Display + Inter
